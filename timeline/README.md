# Timeline System

The timeline is the spine of the project.

The novel begins with Steven waking in **June 2025** and then moves linearly forward in real time. Past events enter only through memory recursion, evidence review, or due diligence.

## Files

| File | Purpose |
|---|---|
| `master_timeline.csv` | Approved canonical timeline events. |
| `timeline_schema.md` | Field definitions and evidence labels. |

## Promotion Rule

Do not add automatically extracted candidates directly into the master timeline without review.

The extractor creates candidates. Steven approves chronology.

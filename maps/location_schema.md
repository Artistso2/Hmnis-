# Location Schema

| Field | Meaning |
|---|---|
| `location_id` | Unique ID. Example: `LOC-0001`. |
| `label` | Author-approved location label. |
| `location_type` | `CITY`, `ADDRESS`, `REGION`, `ROOM`, `ROUTE`, `INSTITUTION`, `UNKNOWN`. |
| `address_or_description` | Address or descriptive location as supplied. |
| `city` | City, if supplied. |
| `state` | State, if supplied. |
| `country` | Country. |
| `latitude` | Approved latitude; blank until verified/approved. |
| `longitude` | Approved longitude; blank until verified/approved. |
| `start_date` | Date associated with timeline. |
| `end_date` | Optional. |
| `source_id` | Source manifest link. |
| `evidence_label` | Evidence label. |
| `verification_status` | Approval/verification status. |
| `public_display` | `yes` or `no`. |
| `notes` | Internal notes. |

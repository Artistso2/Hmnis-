# LaTeX / Overleaf System

This folder is an Overleaf-ready starting point for **Hello, My Name is Steven: The Lived Experience of Habeas Corpus**.

## Files

```text
latex/main.tex
latex/hmnis.sty
```

## Visual Goal

Soft ocean hush + legal precision + source-ledger typography.

The LaTeX edition should feel like:

- a living writ,
- a legal/philosophical document,
- a memoir reconstructed through source evidence,
- an Overleaf-style academic artifact with emotional gravity.

## Overleaf Use

1. Create a new Overleaf project.
2. Upload `main.tex` and `hmnis.sty`.
3. Set compiler to XeLaTeX or LuaLaTeX if you later add custom fonts.
4. Compile.

No external fonts are required in this starter template.

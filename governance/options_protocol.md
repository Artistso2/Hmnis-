# Options Protocol

Steven prefers options and dislikes assumptions.

## Default Editorial Menu

When a meaningful choice arises, present:

| Option | Meaning |
|---|---|
| Safest | Lowest legal/reputational risk; clearest source labeling. |
| Boldest | Strongest artistic/authorial impact; may require review. |
| Hybrid / Recommended | Balances authorial force with legal/editorial precision. |

## When to Ask First

Ask before:

- naming living people publicly,
- changing title/subtitle/brand identity,
- presenting legal conclusions,
- presenting medical conclusions,
- publishing raw sensitive materials,
- changing the chronology spine,
- removing Steven’s words,
- or making a major public-facing push that needs CEO-level sign-off.

## When to Proceed Without Asking

Proceed when the task is:

- structural,
- reversible,
- clearly aligned with existing protocol,
- documentation-only,
- formatting-only,
- or a routine GitHub build/push that does not alter sensitive public claims.

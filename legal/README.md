# Legal / Attorney-Ready Review System

This folder organizes claims, entities, review flags, and attorney-ready packets.

## Important

This repository does not provide legal advice.

These files are designed to make the project rigorous, transparent, source-grounded, and ready for attorney review.

## Core Files

```text
legal/claim_ledger.csv
legal/entity_risk_ledger.csv
legal/publication_review_checklist.md
legal/attorney_packet_template.md
```

## Rule

No public legal conclusion should be stated as adjudicated fact unless supported by an approved source and review status.

Use labels:

- `STEVEN_STATEMENT`
- `SOURCE_EXTRACTED`
- `VERIFIED_DOCUMENT`
- `NEEDS_REVIEW`
- `ATTORNEY_REVIEW_REQUIRED`
- `PRIVATE_ARCHIVE_ONLY`

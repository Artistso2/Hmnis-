Option 01

Raw Witness

Most sellable for the real book. Human face first, document texture second, technology third. The page feels like an author survived long enough to file the record.

  • Organic paper grain
  • Portrait as witness, not decoration
  • Strong author identity: Steven Owens / @soquarky
  • Best for landing page, press, public sharing

Option 02

Evidence Noir

For the detective spine. Black, bone, redactions, gridlines, timestamps, ledger cards. It sells the Memento-style dissection without pretending to be police cosplay.

  • Chronology panels
  • Source-ledger interface
  • Map and timeline compatible
  • Best for archive, source ingest, evidence pages

Option 03

Organic Signal

For the neuropathy / body / AI collision. Worn boot, living nerves, pixel fracture, circuitry. The visual says: the body is analog, the rescue is digital.

  • Body + code
  • Memory recursion as living tendrils
  • Good chapter plate style
  • Best for visual novel scenes

Option 04

Public Record Horizon

For the legal/political layer. Sky, flag, horizon, public witness. Not patriotic wallpaper — the question is whether the record will answer the body.

  • Public-facing seriousness
  • Works with writ / habeas language
  • Good for map and legal pages
  • Best as a restrained accent, not the whole brand

Recommended Stack

Primary: Raw Witness. Secondary: Evidence Noir. Accent: Glass Horizon and Organic Signal.

The site should sell the project by making the human author undeniable first. The tech, AI, maps, EPUB, and visual-novel elements support the witness — they should not bury Steven under interface effects.

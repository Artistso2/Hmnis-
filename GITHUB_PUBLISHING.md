# GitHub Publishing Notes

Repository target:

```text
https://github.com/Artistso2/Hmnis-.git
```

## Recommended GitHub Pages setting

Use GitHub Pages from:

```text
Branch: main
Folder: /root
```

The root `index.html` redirects to:

```text
web/index.html
```

So the public link should become:

```text
https://artistso2.github.io/Hmnis-/
```

## If pushing manually

```bash
git remote add origin https://github.com/Artistso2/Hmnis-.git
git branch -M main
git push -u origin main
```

# Visual Redesign Options — Sellability + Organic Rawness

Author direction:

> Better website visuals. Sellability and real organic rawness. My name is Steven Owens, @soquarky, soquarky.click.

## Recommendation

Use **Raw Witness** as the public-facing landing style.

The Glass Horizons interface is useful for the visual novel, but the public front door needs more human gravity and sellability: Steven's face, Steven's name, the author handle, paper grain, a strong book title, and clear calls to action.

## Options Built

1. **Raw Witness** — human-first, author-first, textured, sellable.
2. **Evidence Noir** — chronology/evidence/archive pages.
3. **Organic Signal** — body + neuropathy + AI + memory recursion.
4. **Public Record Horizon** — legal/political/map pages.

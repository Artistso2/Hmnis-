# Visual Style Guide — Glass Horizons

## Author Direction

The visual feeling for the website and visual novel should follow a **glass horizons** atmosphere:

- something new,
- glass-like,
- luminous,
- personal,
- horizon-facing,
- forensic but not cold,
- digital but still human,
- fragile but not weak.

The supplied images are author-provided visual source material. The portrait images are the author's own face and may be used as personal visual material for the book website and visual novel presentation.

## Visual Keywords

- Glass horizon
- Magenta neon
- Dark violet / near-black field
- Soft cyan horizon light
- Evidence grid
- Memory recursion
- Digital writ
- Body as evidence
- American sky / damaged flag / public record
- Personal portrait / witness / petitioner
- Blueprint / silence / equations / boots / equilibrium

## Palette

| Color | Use |
|---|---|
| Near-black violet | Background / void / confinement |
| Magenta-pink glow | Memory, body, signal, urgency |
| Cyan-blue light | horizon, internet, AI, forward movement |
| Bone white | legal document, testimony, exposed truth |
| Muted graphite | evidence panels, archive cards |
| Warning rose | contradiction, danger, system failure |

## Interface Rules

1. The site should feel like a **visual writ**, not a generic author page.
2. Use glass panels, blur, faint grid lines, and horizon glows.
3. Images should appear as **evidence cards**, **memory cards**, or **signal cards**.
4. Do not use random stock imagery.
5. Do not invent symbolism beyond the author-provided visual direction.
6. The interface must keep the author-control rule visible: **No invented facts. Steven’s words are the source.**
